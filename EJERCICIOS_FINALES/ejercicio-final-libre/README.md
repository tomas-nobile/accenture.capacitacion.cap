# Descripcion

Ejercicio donde se crean nuevos pedidos, que al igual que el anterior, modifican la cantidad de mercaderia.Al crear el nuevo pedido, se crea automaticamente la "fecha de pedido" con la hora y el dia en el que se realiza la peticion, luego se calcula la "fecha de envio" en plazo de 3 dias posterior a la "fecha de pedido", en caso de que sea sabado o domingo se traslada para el lunes.
